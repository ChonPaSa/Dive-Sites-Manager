Create and manage dive sites with some characteristics and a picture. The list can be display on the front-end with a shortcode.


== Description ==
This plugins allows you to easily create a list of dive site on the back-end with basic information like the name, short description and picture. Additionally you can add some information about the dive site, such us, dive type(Drift, Wreck,...), depth, level and so on. 
Once the list is created, it can be displayed on the front-end with just a shortcode.


== Features ==
* Intuitive and easy to use
* Responsive


== Installation ==
From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Dive Sites Manager"
3. **Install** click on "Install Now"
3. **Activate** the plugin when the installation is finished
4. **Click** on the new menu item "Dive Sites" and create your first dive site
5. **Add** the shortcode [cfish-dive-sites] to the page where you want to display the dive site list
